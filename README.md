#Number giving game Java
#About
A simple number guessing game built in Java. The programme generates a random number
between 1 and hundred and the user has to guess it!
#How it works:
1.Programme generates a secret random number
2. User enters a guess
3. Programme says "Too high" or "Too low"
##What I learned
Java syntax and structure, Using Random and Scanner classes,While loops and if/else conditions,Importing Java libraries
